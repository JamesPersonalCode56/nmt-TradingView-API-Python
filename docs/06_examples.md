# 06 - Examples

All runnable examples are in `examples/`.

Common ones:

- `examples/simple_chart.py`
- `examples/quote_session.py`
- `examples/search.py`
- `examples/indicator_basic.py`

Advanced (requires auth):

- `examples/replay_mode.py`
- `examples/custom_chart_type.py`
- `examples/builtin_indicator.py`
- `examples/graphic_indicator.py`
- `examples/all_private_indicators.py`
- `examples/pine_perm_manage.py`

Run examples:

```bash
python examples/simple_chart.py
```

Set auth variables for private endpoints:

```bash
export SESSION=...
export SIGNATURE=...
```
